# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.7.x   | :white_check_mark: |
| < 0.7.x   | :x:                |

## Reporting a Vulnerability

Please report any vulnerabilities to nicolas@seita.nl.

We value your expertise. Howver, currently we have no bug bounty program or similar.
